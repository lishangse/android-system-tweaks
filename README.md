# Android System Development & Optimization

This repository contains various scripts and tools for Android system development, specifically focused on kernel optimization and recovery environment setup for the `haydn` (Redmi K40 Pro) device.

## Projects Included

- **Kernel Tweaks**: Scripts for adjusting CPU governors and I/O schedulers.
- - **Recovery Tools**: Custom scripts for TWRP/OrangeFox environment configuration.
  - - **System Analysis**: Tools for parsing logcat and dmesg outputs for debugging.
   
    - ## Disclaimer
   
    - These tools are intended for advanced users and developers. Use at your own risk. Always ensure you have a full backup before modifying system partitions.
